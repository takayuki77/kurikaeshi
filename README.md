# くりかえし帳

くり返しの家事・メンテナンスのリマインダーと記録アプリ（PWA）。
「メンテ帳」風の機能を、件数制限なし・無料で使えるように自作したもの。

- 公開URL: https://takayuki77.github.io/kurikaeshi/
- iPhoneのSafariで開き「ホーム画面に追加」して使う
- データは端末内（IndexedDB）にのみ保存。バックアップは設定画面の書き出し機能で

## できること

- 周期タスク（5日ごと・2週間ごと・3か月ごと等）と随時タスクの管理
- グループ（掃除・家事・健康・仕事…）で分類し、色付きで表示＋絞り込み
- 前回いつやった？次はいつ？の一覧表示、1タップ完了
- 完了記録にメモ・写真を添付、履歴タイムライン表示
- 予定の延期、過去の記録の追加
- アイコンバッジ（今日やることの件数）
- 書き出し／読み込みによるバックアップ

## 周期の考え方

**開始日 ＝ 次の実施予定日**。開始日を今日にすれば、その日から「今日やること」に出る。
✓で完了すると、**完了した日から間隔ぶん先**が次の予定日になる（7/14に完了・1週間ごと → 7/21）。
既定は「週・1」。

## 構成

- `index.html` … 画面と処理すべて
- `logic.js` … 日付・周期計算（node でテスト可能な純粋関数）
- `sw.js` … オフライン対応＋プッシュ通知の受け口（通知の送信側は第2段階で Cloudflare Workers を予定）
